## Overview

**Workaholic** is a sleek, dark-themed HR dashboard concept featuring a cinematic gate/login experience, animated vault transition, and a fully interactive workforce management interface.

This project is a **showcase demo** — built to demonstrate what Vesa Studios can design and develop. View the full portfolio at [vesastudios.site](https://vesastudios.site).

---

## Features

- 🔐 **Cinematic Gate Screen** — 3D parallax entry with mouse-reactive animations
- 🚪 **Vault Transition** — animated unlock sequence between login and dashboard
- 📊 **Dashboard Home** — KPI stats, team overview, and activity feed
- 👥 **Employees Section** — staff directory with roles and status
- 🕐 **Attendance Tracking** — clock-in/out and daily logs
- 📅 **Leave Requests** — request management and approval flow
- ✅ **Tasks** — task assignment and progress tracking
- 📈 **Analytics** — charts and workforce insights
- ⚡ **Smooth Animations** — powered by Framer Motion throughout

## Getting Started

```bash
# Clone the repo
git clone https://github.com/vedant-B22/workaholic.git
cd workaholic

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
